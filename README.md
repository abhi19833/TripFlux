# TripFlux

TripFlux is a full-stack travel planning application that helps users manage trips, track expenses, and save travel memories in one place.

The project focuses on solving real travel-planning problems with a clean UI and simple workflows.

---

## What You Can Do

- Create an account and log in securely
- Create and manage trips
- Track expenses for each trip
- Add notes and travel memories
- Upload trip photos
- Reset password using email link
- View everything from a single dashboard

---

## Tech Used

**Frontend**
- React
- Tailwind CSS
- Context API

**Backend**
- Node.js
- Express
- MongoDB
- JWT Authentication

**Deployment**
- Vercel (Frontend)
- Render (Backend)
